## How to Generate PDF

### Method 1: Browser Print to PDF (Recommended)
1. Open `KT_DOCUMENTATION.html` in any modern browser (Chrome, Edge, Firefox)
2. Press `Ctrl+P` (Windows) or `Cmd+P` (Mac) to open print dialog
3. Select "Save as PDF" as the destination
4. Choose "A4" paper size and "Portrait" orientation
5. Enable "Background graphics" for better formatting
6. Click "Save" to generate PDF

### Method 2: Using Online Converters
1. Upload `KT_DOCUMENTATION.html` to any online HTML to PDF converter
2. Recommended sites:
   - https://html2pdf.com/
   - https://www.sejda.com/html-to-pdf
   - https://smallpdf.com/html-to-pdf

### Method 3: Using Microsoft Word
1. Open Microsoft Word
2. File → Open → Select `KT_DOCUMENTATION.html`
3. File → Save As → Choose PDF format
4. Click "Save"

### Method 4: Using Adobe Acrobat
1. Open Adobe Acrobat Pro
2. File → Create → PDF from File
3. Select `KT_DOCUMENTATION.html`
4. Click "Create"
